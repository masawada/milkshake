Milkshake - The synchronized Presentation Software
==================================================

Require
-------
Server  
* node.js 0.4.10  
* npm 1.0.22  
* git 1.7.4.4  
Client  
* Web browser(Opera, Firefox, Google Chrome, Safari)  

How To Install it
-----------------
1. `$ git clone git://github.com/masawada/milkshake.git`
2. `$ cd milkshake`
3. `$ npm install express socket.io jade`
4. `$ node app.js`
